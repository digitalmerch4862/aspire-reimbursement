# Aspire Reimbursement

Manual-first reimbursement auditing app with database-backed validation.

## Run Locally

Prerequisite: Node.js

1. Install dependencies:
   `npm install`
2. Configure Supabase keys in `.env`:
   - `VITE_SUPABASE_URL`
   - `VITE_SUPABASE_ANON_KEY`
3. Start the app:
   `npm run dev`

## Build

`npm run build`
