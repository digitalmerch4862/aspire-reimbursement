ASPIRE REIMBURSEMENT TUTORIAL — PREMIERE ASSET PACK
====================================================

FOLDER CONTENTS
  screen.mp4                         The screen recording (silent), 214s, 1260x992
  intro_card.png                     12s title card (full frame, opaque)
  captions/caption_1..8.png          Transparent caption overlays (full frame)
  voiceover/master_voiceover_timeline.wav   All 8 VO lines pre-timed on one 226s track
  voiceover/vo_1..8.mp3              The 8 VO lines as separate clips
  CUE-SHEET.txt                      Exact timecodes for every clip
  README.txt                         This file

QUICK START (5 steps)
  1. Import the whole folder into Premiere.
  2. Right-click screen.mp4 > New Sequence from Clip (gives 1260x992 @ 30fps).
  3. Drag screen.mp4 so it STARTS at 00:00:12:00 on V1.
     Put intro_card.png on V1 from 00:00:00:00 to 00:00:12:00.
  4. Drop master_voiceover_timeline.wav on A1 at 00:00:00:00. Done — audio is timed.
  5. Drop caption_2..8.png on V2 at the IN points in CUE-SHEET.txt.

That's it. Adjust any timing by sliding clips — VO segments are also provided
separately (voiceover/vo_*.mp3) if you'd rather place each line by hand.
